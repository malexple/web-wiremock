Условия на тело запроса (Body Patterns — POST / PUT / PATCH)
Метод DELETE редко используется с телом запроса. Технически допустимо, но некоторые клиенты могут игнорировать тело.
Метод ANY матчит все HTTP-методы включая POST, PUT и PATCH. Body Patterns сработает только если входящий запрос содержит тело.

Все условия объединяются через И. Для OR — создайте отдельный стаб с другим условием.
